# Claude Morning Greeting — 远程代理提示词

本文件供 Claude Code 远程定时代理（/schedule）使用。
创建 trigger 时，将下方提示词作为 agent 的输入即可。

---

## 推荐提示词

```
你好，请用一句话回复即可。不需要读取或修改任何文件。
```

## 为什么这样写

远程代理启动时拥有完整的工具集（Bash、Read、Write 等）和 Git 仓库访问权限。
如果提示词太模糊（如只说"你好"），代理可能会：

- 尝试读取仓库文件来"理解上下文"
- 执行不必要的命令
- 消耗额外的 token 和时间

加上"不需要读取或修改任何文件"这一约束，确保代理只做一件事：**回复一句话**，
从而以最小的开销触发 5 小时用量窗口的重置。

## 备选提示词

根据场景选择：

### 极简版（最省 token）

```
回复"收到"两个字即可，不要执行任何工具。
```

### 带时间感知版

```
现在几点了？用一句话回复当前时间，不需要读取或修改任何文件。
```

### 英文版

```
Reply with a single short sentence. Do not read or modify any files.
```
